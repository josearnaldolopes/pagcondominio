Sobre

A API do novo jeito de pagar seu condomínio

É uma fintech de meios de pagamentos que possibilita os condôminos a optarem por uma forma de pagamentos, seja por cartão de credito, cartão de debito, Debito automático ou boleto, facilitando o dia dia do condomínio e reduzindo a inadimplência. Simples e fácil, Já nascemos plugados com grandes players do mercado como adquirentes e seguradoras internacionais.

Credenciamento de condomínio

Credenciar o condomínio existe duas formas:
1. Através do envio dos dados de CPF ou CNPJ, endereço completo, conta bancária e responsável para contato@pagcondominio.com
2. Ou através de nossas Apis abaixo.
Dentro de 7 dias, o condomínio estará apto a receber por cartão de crédito e será enviado um email de confirmação

URL ENDPOINT

GET | POST

https://api.pagcondominio.com/v1/erp/condominio/token/{{UsuarioERP}}

header 'accept: application/json'

header 'content-type: application/json'

Json da requisição 
  [
    {
     "condominio": "Nome do Condomínio",
     "administradora": "Nome da administradora",
     "cep": "CEP do endereço",
     "logradouro": "Endereço",
     "numero": "Número do logradouro",
     "bairro": "Bairro do logradouro",
     "cidade": "Cidade do bairro",
     "estado": "Sigla do Estado",
     "documento": "CPF ou CNPJ",
     "nomebanco": "Nome do banco",
     "numeroagencia": "Número da agência",
     "numeroconta": "Número da conta"
    }
  ]
Json de resposta com sucesso 
  [
    }
     "status": "Cadastro de condominio efetuado com sucesso"
    }
  ]
Json de resposta com erro 
  [
    }
     "status": "Ocorreu um erro no cadastro de condominio"
    }
  ]

Obter dados do condomínio

Uma vez aprovado e com a chave em mãos é possivel ver os dados do condominio

URL ENDPOINT

GET

https://api.pagcondominio.com/v1/erp/condominio/token/{{UsuarioERP}}/chave/{{TokenCondominio}}

header 'accept: application/json'

header 'content-type: application/json'

Json da resposta 
  [
    }
     "condominio": "Nome do Condomínio",
     "administradora": "Nome da administradora",
     "cep": "CEP do endereço",
     "logradouro": "Endereço",
     "numero": "Número do logradouro",
     "bairro": "Bairro do logradouro",
     "cidade": "Cidade do bairro",
     "estado": "Estado da cidade",
     "documento": "CPF ou CNPJ",
     "nomebanco": "Nome do banco",
     "numeroconta": "Número da conta",
     "taxavista": "Valor em porcentagem (ex: 0.5%)",
     "taxacredito": "Valor em porcentagem (ex: 4%)",
     "status": "Aprovado para produção"
    }
  ]

Cadastro de condomíno

Uma vez credenciado e liberado o condomínio, é possivel manipular os condominos de cada condomínio.

URL ENDPOINT

GET | POST

https://api.pagcondominio.com/v1/erp/condominio/token/{{UsuarioERP}}/chave/{{TokenCondominio}}

header 'accept: application/json'

header 'content-type: application/json'

Json da requisição 
  [
    }
     "condomino": "Nome do Condomíno",
     "apartamento": "Número do Apartamento",
     "bloco": "Bloco do Apartamento",
     "documento": "CPF ou CNPJ",
     "nome": "Razão Social ou nome",
     "valor": "Valor em centavos",
     "cartao": "Número do cartão",
     "nomecartao": "Nome como gravado no cartão",
     "validade": "Validade gravada no cartao",
     "vencimento": "Dia do vencimento",
     "cvv": "CVV do cartão",
     "cpf_2": "CPF do segundo Condomíno [Opicional]",
     "cartao_2": "Numero do segundo cartão [Opicional]",
     "nomecartao_2": "Nome como gravado no segundo cartão [Opicional]",
     "validade_2": "Validade gravada no segundo cartão [Opicional]",
     "vencimento_2": "Dia do vencimento segundo cartão [Opicional]",
     "cvv_2": "CVV do segundo cartão [Opicional]",
     "recorrencia": true ou false,
     "parcela": "0 [Máximo 3 parcelas. Se 0 ou vazio equivale a parcela única]",
     "valorparcela": "Valor da parcela em centavos",
     "salvar": true ou false
    }
  ]
Json de resposta com sucesso (salvando dados para futuras transações) 
  [
    }
     "token": "[String do token]",
     "status": "Cadastro de condomino efetuado com sucesso"
    }
  ]
Json de resposta com sucesso (pagamento) 
  [
    }
     "comprovante": "[String do comprovante]",
     "status": "Cadastro de condomino efetuado com sucesso"
    }
  ]
Json de resposta com erro 
  [
    }
     "status": "Ocorreu um erro no cadastro de condomino"
    }
  ]
